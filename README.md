# Agnes-Claw Risk & Protection

Agnes-Claw Risk & Protection is a React + TypeScript + Vite demo that lets you paste a draft social media post, run five separate Agnes-Claw audience agents against it in parallel, review a PR-risk synthesis before publishing, and then launch an autonomous protection pipeline that monitors comments and can decide whether to take the post down. Copy `.env.example` to `.env`, set `VITE_AGNES_BASE_URL`, `VITE_AGNES_API_KEY`, and `VITE_AGNES_MODEL`, then install dependencies and run the app with `npm install` followed by `npm run dev`.
