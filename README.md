# Pre-Post Reaction Simulator

Pre-Post Reaction Simulator is a React + TypeScript + Vite demo that lets you paste a draft social media post, run five separate Agnes-Claw audience agents against it in parallel, and review a final PR-risk synthesis before publishing. Copy `.env.example` to `.env`, set `VITE_AGNES_BASE_URL`, `VITE_AGNES_API_KEY`, and `VITE_AGNES_MODEL`, then install dependencies and run the app with `npm install` followed by `npm run dev`.
